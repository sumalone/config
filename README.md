## Setup `tmux`

1. copy or symlink the `tmux` config directory to `~/.config/tmux`

## Setup `tpm - Tmux Plugin Manager`

1. Clone the repo
`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`

2. Install plugins
Start tmux and run `<prefix> + I`
